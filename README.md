# JS_nomad_painting
# JavaScript <canvas API> 활용해 그림판 만들기
  
- JavaScript의 mouse이벤트를 활용하여 그림판을 만들어보았다.
- canvas API사용  
- 강의기능 구현 : 두께 조절, 색상 선택, 파일저장
- 파일 저장 : canvas에 그려진 data를 URL로 변환시켜, a태그의 href, download메소드를 활용해 파일로 변환시켜 저장도 가능하다
